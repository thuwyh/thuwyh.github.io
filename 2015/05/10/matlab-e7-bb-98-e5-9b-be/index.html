Matlab绘图

发表于 2015-05-10   |   分类于 Code4Fun   |  

matlab 是我在日常科研和学习中常用的数据处理和仿真工具，自然也会经常用到他的绘图功能。matlab的绘图功能其实是很强大的，但是初学者往往不能很好掌握。其实不仅是技术上的问题，以我对身边同学的观察，我们对图表的要求和处理图表的能力和优秀期刊文章的水平相差实在太远了。之前知乎就有一个问题：

如何在论文中画出漂亮的插图？
第一次看的时候真的被里面漂亮的图画震撼到了，但那个回答主要是介绍matplotlib的，毕竟是python上的东西，用起来不如matlab顺手，于是之前就花了点时间学习了matlab绘图，今天将一些经验在这里分享。

1. 规范

画图，不管用什么工具，首先要时刻牢记保持图的规范性。一张规范的图需要有以下几个部分：

  1. 标题
  2. 坐标轴、坐标轴的标题、单位及刻度
  3. 图例
  4. 其他辅助性的文本和标注
    对于期刊文章中使用的图，标题一般不出现在图中，而是题注在图下方，而且必须包含图片编号。而且通常在题注中要对图有基本的说明，可以不看正文就看懂图的表面含义。付老师常说好的图要是自含式的，通过图就能清晰地将你要揭示的规律展示出来，这当然又是更高的要求。不过画图之前确实应该精心谋划一下，例如坐标轴的选择，图的形式等等。

2. 图线、坐标轴及图例

首先推荐大家使用matlab中的句柄，可以方便地修改图的参数。获得句柄非常简单，例如

L1=plot(x/1e3, y_dis/1e6);
l1就是句柄了，相当于给图线取了一个名字，方便修改。 有了句柄我们就可以设置图线的各种属性了，比较常用的属性有linewidth、linestyle、color。从名字看，他们的意义都非常明了了。其中linestyle有几种常用的 * - ：实线 * -- ：虚线 * -. ：点划线 color我想重点说下，matlab自带的颜色比如rgb、cmyk都太丑了，应该果断抛弃。其实我们是可以自定颜色的，方法很简单，给出相应的rgb分量值就行了（不同于0-255整数表示，matlab是用0-1浮点数表示的）。可以到网上搜到大量的配色方案，这样你就可以画出逼格比较高的图了，例如 
set(l1,'color',[216,34,13]./255);
图线说完了说说坐标轴，可以直接通过句柄gca操作当前坐标轴。坐标轴的常用属性有*tick, *lim，*代表x或y。前者用来指定显示的坐标刻度，后者用来指定坐标轴范围。大家用matlab里试一下就清楚了，例如 
set(gca,'xTick',[0,5,10]);
set(gca,'xlim',[-1,11]);
matlab还有很强大的一点是他的坐标轴label以及图例legend中的文字可以解析latex。这样你就可以方便地表示\(\theta\),\(\alpha\)之类的特殊符号以及分式等比较复杂的东西，置于latex语法大家可以到网上搜，结合[在线latex编辑器](http://latex.codecogs.com/eqneditor/editor.php)就能很方便地练习和尝试了。说到文字就不得不提一下字体和字号的设置，这些可以用fontsize, fontname属性来搞定，例如 
set(gca,'fontname','Times New Roman','fontsize',13);
或者直接在输出时指定 
xlabel('\fontname{TimesNewRoman}\fontsize{13}\it\theta \rm(^{\circ})');
ylabel('\fontname{TimesNewRoman}\fontsize{13}\ity');

3. 尺寸控制及输出

大部分人用matlab画图是不控制尺寸的，画出来贴到word里再调，这是很不好的。最大的弊端就是你无法准确判断图经过缩放后的效果（特别是字的大小和线的粗细），造成返工浪费时间的情况。所以应该在画图的时候就规划好，而且这在正式场合是容易做到的，你应该很清楚你的图是占整页例如16cm宽还是分两栏中的一栏如8cm宽。使用下面的代码就可以修改图的大小了

width=8;
height=8;
set (gcf,’windowstyle’,’normal’);
set (gcf,’units’,’centimeters’,’Position’,[0,0,width,height]);

其中的width和height是以厘米为单位的，你可以随意定制。

做好图之后还有一道重要的工序就是输出。请不要使用截图工具！！！！！那样真的太不专业了，另外我也不建议使用copyfigure，我用的时候总有一些问题，比如线有锯齿之类的。在这里强烈推荐大家一个插件export_fig，它是一个开源项目，具体使用方法请移步项目首页 https://github.com/altmany/export_fig

好了，我要说的说完了，给大家提供一个模板，你可以方便地用于自己作图。也可以下载我在某次讲座中的材料，那里除了介绍matlab画图还涉及到一些matlab GUI相关知识。

% ————————————————-
% coded by wyh
% see more detail on www.wuyuanhao.com
% ————————————————-
%数据
x=0:0.001:10;
y1=sin(x);
y2=sin(2x)+sin(3x);
%画图
figure(1)
hold on;
l1=plot(x,y1);
l2=plot(x,y2);
%线宽
set(l1,’linewidth’,1.5);
set(l2,’linewidth’,1.5);
%颜色
set(l1,’color’,[216,34,13]./255);
set(l2,’color’,[88,76,157]./255);
%线形
set(l1,’linestyle’,’–’);
%坐标轴
set(gca,’fontname’,’Times New Roman’,’fontsize’,13);
set(gca,’xTick’,[0,5,10]);
set(gca,’xlim’,[-1,11]);
set(gca,’xgrid’,’on’);
%坐标轴标题
xlabel(‘\fontname{TimesNewRoman}\fontsize{13}\it\theta \rm(^{\circ})’);
ylabel(‘\fontname{TimesNewRoman}\fontsize{13}\ity’);
%标注
legend(‘y1’,’y2’);
%尺寸
width=8;
height=8;
set (gcf,’windowstyle’,’normal’);
set (gcf,’units’,’centimeters’,’Position’,[0,0,width,height]);
%输出
export_fig fig1.tiff -r600 -painters -transparent
