自定义wordpress主题

发表于 2015-04-19   |   分类于 Code4Fun   |  

原来采用官方提供的twentyten主题，感觉有改进空间，就自己修改了下，主要是给首页文章增加了背景色，让文章间区分更显著。

相比于直接修改主题style.css文件的方法，wordpress可以建立子主题，这样可以防止升级主题后你的修改被覆盖。

建立子主题很容易，可以按照下面的步骤

1、首先在主题文件夹下(wp-content\themes)建立一个文件夹，用来存放子主题的所有文件，例如：twentytwelve-child。

2、在twentytwelve-child文件夹下新建一个style.css文件，该文件是必须的。将以下代码粘贴进去：

/
Theme Name:     （子主题名称，必须）
Theme URI:      （主题url）
Description:    （主题描述）
Author:         （主题作者）
Author URI:     （作者uri）
Template:       （父主题目录名称，必须）
Version:        （版本号）
/

将括号内容替换成你的信息，接下来修改你想修改的样式部分就行了。

我对.hentry的修改如下

/
Theme Name:     Twenty Twelve Child
Theme URI:      http://www.wuyuanhao.com
Description:    Child theme for the Twenty Ten theme
Author:         wuyuanhao
Author URI:     http://www.wuyuanhao.com
Template:       twentyten
Version:        1.0.0 
/

@import url(“../twentyten/style.css”);

.hentry {
margin: 0 0 18px 0;
background-color: rgba(220, 220, 220, 0.22);
padding: 10px 15px 15px 15px;
border-radius: 5px;
}
之后你就可以在“主题”面板里启用你建立的自主题啦！
